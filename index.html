Symbols Description

  • AC: Clear all
  • DEL: Delete last character
  • ÷, ×, −, +: Division, Multiplication, Subtraction, Addition
  • π: Pi constant (~3.14159)
  • e: Euler's number (~2.718)
  • xʸ (^): Power function
  • √: Square root
  • sin, cos, tan: Trigonometric functions
  • log: Logarithm base 10
  • ln: Natural logarithm
  • ±: Change sign
  • PV, FV, PMT, NPV: Financial calculations (Present value, Future value, Payment, Net present value)
  • Σ+, Σ-: Summation plus/minus
  • s: Sample standard deviation
  • R/S: Range/Standard deviation
  • GTO, BST, PSE: Programming/statistical functions
